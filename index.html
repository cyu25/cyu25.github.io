    Projects

    Flappy Bird Game

    Inspired by the app "Flappy Bird," this game is played by controlling a player with the spacebar. The objective is to navigate through as many warp pipes as possible. If the character touches one of the pipes, the game ends, and therefore, restarting the gameplay. This project was created using Python.

    Morse Code Converter

    Implemented with VHDL Code, the Basys3 FPGA board was used to generate Morse code signals by lighting a LED and producing sounds. A user will type letters/numbers into a keyboard, and the characters will take the form of 8-bit ASCII characters to be sent out as Morse Code signals. In the appendix, the source code is demonstrated.

    3D Modeling Designs

    A custom environment, an abstract image, and a biped character were designed using Autodesk Maya. Each design was modeled and textured to demonstrate a more "real-life" feel. A video is also attached to include the animation of the biped.

    Marble Maze

    A full control system for a marble maze was designed using C. The maze was designed with the functionality of stepper motors, ensuring that the maze responded smoothly to the joystick movements. Additionally, the code controlled the game with start/stops states from photo sensors, game timer countdown, winning and losing results, and respective LED and Seven Segment displays.

    Sudoku Game

    This game uses a stack-based solver to implement a search algorithm. The objective of Sudoku is to complete a 9x9 grid with numbers from 1-9, without repeating each number in a column, row, and their local 3x3 squares. The game was created using Java.
